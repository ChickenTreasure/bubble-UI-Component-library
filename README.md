# bubble-UI 组件库

#### 介绍
bubble-UI组件库的环境搭建：
  技术选型： vue3 + typescript + babel + sass + vite;
  代码提交:    husky统一git提交检验;
  代码风格:    eslint + prettier;
  文档开发:    vitepress;

#### 软件架构
软件架构说明

版本说明： 
    pnpm版本  7.25.0
    node版本  16.18.0

#### 环境运行教程
打开终端，输入以下命令:
1.  pnpm install
2.  pnpm run dev
